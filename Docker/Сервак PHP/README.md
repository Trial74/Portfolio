# Сервер PHP — Docker контейнер для тестирования

Простой Docker-контейнер на базе PHP 8.3 для различных тестов и экспериментов с PHP.

## Описание

Контейнер включает:
- **PHP 8.3 CLI** — встроенный веб-сервер
- **Xdebug** — для отладки кода
- Автоматический запуск встроенного PHP сервера на порту 8000

## Быстрый старт

### Запуск контейнера
```bash
docker compose up -d --build
```

### Остановка контейнеров
```bash
docker compose stop
```

## Использование

После запуска контейнера приложение будет доступно по адресу:
- **http://localhost:8090**

Исходный код находится в папке `src/` и автоматически монтируется в контейнер.

## Отладка с Xdebug

Контейнер настроен для работы с Xdebug:
- Порт отладчика: **9003**
- Режим: автоматический запуск при запросе
- Хост: `host.docker.internal` (для подключения из IDE на хосте)

## Структура проекта

```
.
├── Dockerfile              # Образ контейнера
├── docker-compose.yml      # Конфигурация Docker Compose
├── xdebug.ini             # Настройки Xdebug
└── src/
    └── index.php          # Тестовый PHP файл
```

## Технические детали

- **PHP версия:** 8.3
- **Внутренний порт:** 8000
- **Внешний порт:** 8090
- **Рабочая директория:** `/var/www`
