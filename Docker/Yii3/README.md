# Yii3 — Docker контейнер для обучения

Docker-окружение для изучения и разработки на фреймворке **Yii3**.

## Описание

Полнофункциональное окружение разработки включает:
- **PHP 8.3-FPM** с расширениями для работы с БД
- **Nginx** — веб-сервер
- **MySQL 8** — реляционная БД
- **PostgreSQL 15** — альтернативная БД
- **Xdebug** — отладка кода
- **Composer** — менеджер зависимостей

## Быстрый старт

### Запуск контейнеров
```bash
docker compose up -d --build
```

### Остановка контейнеров
```bash
docker compose stop
```

## Доступ к сервисам

- **Приложение:** http://localhost:8080
- **MySQL:** localhost:3306
  - База данных: `yii3_db`
  - Пользователь: `yii3_user`
  - Пароль: `yii3_pass`
- **PostgreSQL:** localhost:5432
  - База данных: `yii3_pg_db`
  - Пользователь: `yii3_pg_user`
  - Пароль: `yii3_pg_pass`

## Отладка

Xdebug настроен для работы с IDE:
- Порт: **9003**
- Хост: `host.docker.internal`

## Структура проекта

```
.
├── docker-compose.yml    # Конфигурация сервисов
├── Dockerfile           # Образ PHP
├── nginx/
│   └── default.conf     # Конфигурация Nginx
└── app/                 # Приложение Yii3
    └── .env            # Переменные окружения
```
